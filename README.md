# PopX App

A pixel-perfect React JS implementation of the PopX mobile app UI — 4 screens built from an Adobe XD design.

## 🖥️ Live Demo

[Hosted on Vercel](https://your-vercel-link.vercel.app)

## 📂 Pages

| Page | Route | Description |
|------|-------|-------------|
| Landing | `/` | Welcome screen with Create Account & Login |
| Login | `/login` | Sign in with email & password |
| Sign Up | `/signup` | Create a new PopX account |
| Profile | `/profile` | Account settings & user info |

## 🛠️ Tech Stack

- **React JS** (Create React App)
- **React Router DOM v6** — client-side navigation
- **Vanilla CSS** — pixel-perfect styling, no UI library
- **Google Fonts** — Inter typeface

## 🚀 Getting Started

```bash
npm install
npm start
npm run build
```
